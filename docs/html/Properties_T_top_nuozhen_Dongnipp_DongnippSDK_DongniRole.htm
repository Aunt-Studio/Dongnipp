      DongniRole 属性

      属性

      ClassId 角色所属的班级的 ID
      ClassName 角色所属的班级的名称
      GradeId 角色所属的年段的 ID
      GradeName 角色所属的年段的名称
      RelativeId 角色亲戚ID，例如该角色为家长，则该值为家长角色的 ID
      RoleSort 角色的序号
      SchoolId 角色所属的学校 ID
      SchoolName 角色所属的学校名称
      StudentId 角色关联的学生 ID
      StudentName 角色关联的学生姓名
      User 角色所属的 DongnippSDKDongniUser 实例
      UserType 角色类型。
